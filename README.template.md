# nautical-charts

![npm](https://img.shields.io/npm/v/nautical-charts)

A pure JavaScript library for reading nautical chart files. In particular, for reading BSB-formatted (`.bsb` and `.kap`) files.

## Installation

```bash
$ npm install nautical-charts --save
```
